01 — 파일
~ (분석할 피크가 들어있는 대략적 범위 — 여기서 피크를 찾은 뒤, 실제 피팅 창은 피크 중심 ± ΔE(eV)로 자동 설정됩니다. 비우면 전체에서 자동 감지)
엑셀 파일(.xlsx/.xls)은 두 형식을 자동 인식합니다: ① A열=공통 파장, B열부터 각 열이 데이터 하나 ② 열 쌍(A·B=데이터1, C·D=데이터2 ...). 첫 행이 숫자가 아니면 이름(헤더)으로 사용합니다.
파일 옆 min~max 칸은 그 파일만의 "피크 탐색 구간"입니다 (피크를 어디서 찾을지). 실제 피팅 창은 모든 파일 동일하게 피크 중심 ± ΔE(eV)로 자동 설정되므로, 이 칸은 엉뚱한 피크(아티팩트)를 잡을 때만 좁혀주면 됩니다. 비워두면 위 기본 탐색 구간(또는 자동 감지)을 따릅니다.
